Asbestos Materials/Uses

Asbestos has been used within over 4,000 products, the vast majority of which were building materials. These include the following asbestos containing materials (ACM’s):

  • Loose Fill
  • Sprayed asbestos
  • Thermal Insulation
  • Asbestos Insulation Board
  • Asbestos Cement
  • Textured Coatings
  • Textile/Paper/Insulation/other products
